# Security Compliance

GrowthBook takes security seriously and is compliant with **SOC2**, **GDPR**, and **HIPAA**. We perform regular penetration testing, have an active bug bounty program, and maintain strict controls to protect data privacy.

View our [Trust Center](https://trust.growthbook.io/) to request access to the above ceritifications, as well as our most recent penetration test report, incident response plan, access control policy, and more.
